### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of how to open HTML files in a browser
- (Optional) A code editor to view or modify the source code

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/GitKoshu.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd GitKoshu
   ```

3. **Open the game**
   - Double-click on `index.html` to open it in your default web browser
   - Or, right-click on `index.html` and select "Open with" to choose your preferred browser

4. **Start playing!**
   - Click on any of the three options (Rock, Paper, or Scissors) to make your move
   - The computer will automatically make its move
   - The score will update based on the result of each round
